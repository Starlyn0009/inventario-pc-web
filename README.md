# Sistema de Inventario - Tienda de Computadoras

## Descripción

Sistema web desarrollado para la gestión de inventario de productos en una tienda de computadoras.

##  Tecnologías

* HTML, CSS, JavaScript
* Supabase (PostgreSQL)
* GitHub

##  Funcionalidades (Primer Release)

* Crear productos
* Listar productos
* Editar productos
* Eliminar productos

##  Metodología

Este proyecto se desarrolla utilizando metodología Scrum con entregas incrementales.
